## Online Store

Учебный интернет-магазин на Django: каталог товаров с категориями и изображениями, складские остатки, корзина, оформление заказа и история покупок пользователя.

### Возможности

- регистрация и вход пользователя;
- каталог товаров и фильтрация по категориям;
- загрузка изображений товаров с хэшированием имени файла;
- корзина с суммированием одинаковых товаров и списанием складского остатка;
- оформление заказа из корзины;
- история заказов с сохранением снимка названия, цены и количества товара;
- management-команды для импорта товаров и экспорта остатков.

### Запуск через Docker

1. Скопируйте пример переменных окружения:

```bash
cp .env.example .env
```

2. При необходимости измените значения в `.env`.

3. Соберите и запустите контейнеры:

```bash
docker compose up --build
```

Приложение будет доступно на `http://localhost:8000/`.

### Management-команды

Загрузить товары из CSV или JSON:

```bash
python manage.py load_goods path/to/goods.csv
```

Экспортировать остатки товаров:

```bash
python manage.py export_product_residue path/to/residue.csv
```

### CSV для импорта

Минимальные колонки: `name`, `description`, `price`, `category`, `quantity`.
